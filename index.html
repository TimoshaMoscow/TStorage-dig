О проекте

Хранилище с мета-тегами - это современная платформа для организации и управления цифровыми ресурсами.

Основные возможности:

  • Расширенная система мета-тегов для точной категоризации
  • Адаптивный интерфейс с поддержкой всех устройств
  • Мощный поиск по всем атрибутам элементов
  • Интуитивное управление коллекциями ресурсов

Добавление нового элемента

Как добавить элемент в локальную версию:

1
Скопируйте шаблон

Перейдите во вкладку "Шаблон" и скопируйте готовый код

2
Настройте данные

Замените плейсхолдеры на вашу информацию

3
Добавьте в код

Вставьте элемент в массив items в файле index.html

4
Сохраните изменения

Обновите страницу чтобы увидеть новый элемент

💡 Важная информация:

Этот функционал работает только в локальной версии сайта. Изменения будут видны только вам.

Чтобы добавить элемент на основной сайт - свяжитесь с администратором через вкладку "Контакты".

Готовый шаблон для копирования:

Код элемента
{ id: items.length + 1, title: "Название вашего ресурса", description: "Описание вашего ресурса", image: "images/ваше-изображение.png", tags: ["тег1", "тег2", "тег3"], url: "https://ссылка-на-ресурс.com" }

📖 Пояснения к полям:

id Уникальный номер (автоматически увеличивается)
title Название ресурса (максимум 60 символов)
description Краткое описание (рекомендуется 120-180 символов)
image Путь к изображению (формат: PNG/JPG, размер: 280×180px)
tags Массив тегов для поиска (3-5 релевантных тегов)
url Действующая ссылка на ресурс (обязательно https://)
