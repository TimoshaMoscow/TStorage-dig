AutoHE - Визуальный редактор для Minecraft

AutoHE - это визуальный редактор для создания игровых историй в Minecraft. Создавай крутые проекты без программирования. Возможности:

• Визуальное программирование через Blockly-блоки

• Мгновенный экспорт проектов в Minecraft

• GUI редактор для NPC диалогов

• Менеджер ресурсов (модели, текстуры, звуки)

• Темная тема с настраиваемыми акцентами

• Локальное хранение проектов

Процесс создания: скачай → создай проект → собери логику из блоков → экспортируй в игру. Доступно для Windows, Linux, macOS.
